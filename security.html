Security

Last updated: July 2026

At Leads BPO, we take the security of your data seriously. This page describes the practices we have in place to protect the information you share with us through our website forms.

Our Security Practices

lock

Encrypted Connections

All data transmitted between your browser and our website is encrypted using HTTPS/TLS. This means your form submissions travel securely and cannot be read by third parties in transit.

shield

Row Level Security (RLS)

Form submissions are stored in a database protected by Row Level Security policies. Anonymous visitors can only submit data — they cannot read, modify, or delete any records. Admin access requires authenticated credentials.

key

Secure Admin Access

Our admin panel requires authenticated login through Supabase Auth. Admin accounts are created manually — there is no public registration. Session tokens are managed securely.

bug_report

Spam & Bot Protection

Our forms include server-side validation, honeypot fields, minimum submission timing checks, and submission rate limiting to reduce spam and automated bot submissions.

manage_accounts

Minimal Data Collection

We collect only the information necessary to respond to your enquiry or application. We do not collect payment information, store passwords, or require account creation for visitors.

visibility_off

No Secret Keys in Browser Code

Our website uses only a public anon key in browser-side code — never a service-role key or database password. All sensitive operations are protected by server-side security policies.

Limitations

While we apply reasonable measures to protect your information, no internet transmission or electronic storage system is entirely free from security risks. We encourage you to use up-to-date browsers and to avoid submitting sensitive information (such as passwords or financial details) through general contact forms.

Reporting a Security Issue

If you believe you have found a security vulnerability on our website, please contact us responsibly at leadsbpo65@gmail.com with a clear description of the issue. We will investigate and respond promptly.
